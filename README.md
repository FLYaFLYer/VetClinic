# Курсовой проект

## Тема: ветеринарная клиника.

### Пояснение
Данный проект делается не для публичного использования, а для успешной сдачи курсового проекта и получения личного опыта.
В данном проекте будет простое приложение для пользователя, т.е. для работника, ну и для администратора.

### Роли:
- Работник: следит за лекарствами, записывает в БД пациентов, хозяинов животных, общую информацию о них и т.п.
- Администратор: следит за аккаунтами, делает с ними манипуляции, создаёт новые аккаунты работникам, имеет полный доступ ко всему приложению

### Используемые ресурсы

Делался данный проект в этих приложениях: **"Visual Studio 2022"** и **"SQL Server Management Studio 2022**.
ЯП в данном проекте использовались такие: **C#**, **SQL**, **XAML**.
Так же использовались такие технологии, как: **WPF**, **.NET Framework 4.8.1**.
